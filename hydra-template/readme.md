# Chat Channel Location Shape (Hydra Template Approach)

## Overview

This model defines how a mee:LongChat storage location is represented using a Hydra URI template with explicit RDF variables.

It separates:

* Chat resource (mee:LongChat)
* Location template (hydra:IriTemplate)
* Explicit derived properties (ex:year, ex:month, ex:day)
